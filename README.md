# Offset Printing Services Website

This is a modern, responsive website built to showcase offset printing services such as diaries, notebooks, registers, and other custom print works.  
The website is intended to be shared with clients to view available services, pricing details, and service availability.

This is not an e-commerce platform. Orders and payments are handled offline.

---

## Tech Stack

- Vite
- TypeScript
- React
- Tailwind CSS

---

## Purpose

- Display offset printing services
- Share pricing and availability with clients
- Provide a clear overview of printing capabilities
- Act as a digital catalogue for the business

---

## Services Included

- Diaries and planners
- Notebooks and notes
- Registers
- Visiting cards
- Custom offset printing
- Bulk printing services

---

## Features

- Responsive design for all devices
- Clean and professional UI
- Fast performance using Vite
- Easy to update service and pricing content
- Lightweight and client-friendly

---

## Project Setup

### Clone the repository
```bash
git clone <repository-url>
cd <project-folder>
```

---

### Install dependencies
```bash
npm install
```

---

### Run development server
```bash
npm run dev
```

---

### Build for production
```bash
npm run build
```

---

## Styling

This project uses Tailwind CSS for styling.

- No external UI frameworks
